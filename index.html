YouTube HTML5 Video Mirroring Bookmarklet

About

If your browser supports HTML5 video and it's enabled on YouTube, you can use this bookmarklet to instantly flip HTML5 videos horizontally, which can create "mirrored" versions of dance videos to learn from.

How to use

Drag this link to your bookmarks bar:

Mirror/Unmirror

When you are watching an HTML5 video on YouTube, click the Mirror/Unmirror button you created to mirror the video. Alternatively, you can right click on the link and add it to your bookmarks, and open the bookmark when you're watching an HTML5 YouTube video.

You can tell if a video is HTML5 by right-clicking on the video: If you see "About the HTML5 player" then it's HTML5.

Sometimes videos viewed from a channel won't be HTML5, but if you open the video's page ("View comments, related videos, and more") it will be HTML5.

To find videos that can be mirrored, you can select WebM under Features in YouTube's Advanced Search options.

Note that you have to join the HTML5 trial each time you use a different computer/browser.

Source code

(function(){
    var numberOfVideos = document.getElementsByTagName("video").length;
    
    if(numberOfVideos >= 1)
    {
        var video;
        for(var i = 0; i < numberOfVideos; ++i)
        {
            if(document.getElementsByTagName("video")[i].videoHeight > 100)
            {
                video = document.getElementsByTagName("video")[i];
                break;
            }
        }
        if(video.style.cssText == "")
        {
            window.mirrored=true;
            video.style.cssText = "-moz-transform: scale(-1, 1); -webkit-transform: scale(-1, 1); -o-transform: scale(-1, 1); transform: scale(-1, 1); filter: FlipH;";
            return;
        }
        
        if(window.mirrored)
        {
            video.style.cssText = "-moz-transform: scale(1, 1); -webkit-transform: scale(1, 1); -o-transform: scale(1, 1); transform: scale(1, 1);";
            window.mirrored = false;
        } 
        else if(!window.mirrored)
        {
            video.style.cssText = "-moz-transform: scale(-1, 1); -webkit-transform: scale(-1, 1); -o-transform: scale(-1, 1); transform: scale(-1, 1); filter: FlipH;";
            window.mirrored = true;
        }
    };
})();
