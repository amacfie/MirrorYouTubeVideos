YouTube Video Mirroring Bookmarklet

About

You can use this bookmarklet to instantly flip videos horizontally, so you can create "mirrored" versions of dance videos (or any other videos) to learn from.

How to use

Drag this link to your bookmarks bar:

Mirror/Unmirror

Updated August 12, 2024

When you are watching a video on YouTube, click the Mirror/Unmirror button you created to mirror the video. Alternatively, you can use the "Bookmark manager" to create a bookmark, and then open the bookmark when you're watching a YouTube video.

Using a mobile device? You can use the bookmarklet on youtube.com in the web browser, e.g. Chrome in Android.
